## Image Handling Rules
1. Only use images present in `D:\Projects\images`
2. Extensions are `.jpg` (clean up `.JPG.jpeg` on copy)
3. Hero uses `background-image` for proper face visibility
4. Service cards use `object-fit: contain` for full image display
5. Gallery uses `object-fit: cover` with `object-position: center top`
6. Never reference images that don't exist (causes broken images)
